# SmartSpend Tracker 💸

Умный SPA-трекер личных финансов с автоматической категоризацией транзакций через ИИ (Google Gemini API). Приложение помогает визуализировать расходы, планировать бюджеты и управлять накоплениями.

## 🚀 Основные возможности (Features)

### 🧠 AI-Категоризация
Приложение использует модель `gemini-3-flash-preview` для анализа текстового описания траты. Система автоматически подбирает наиболее подходящую категорию из пользовательского списка и назначает контекстный эмодзи.

### 📊 Аналитика и Визуализация
*   **Пузырьковая диаграмма (Bubble Chart):** Наглядное представление категорий, где размер пузырька пропорционален сумме трат.
*   **Динамика трат:** Столбчатая диаграмма расходов по дням месяца с разделителями недель.
*   **Интерактивный календарь:** Быстрая навигация по дням, отображение ежедневных итогов и меток доходов (ЗП/Аванс).
*   **Селектор периода:** Быстрый выбор любого месяца и года через модальное окно.

### 🎯 Управление целями
*   **Проекты и События:** Создание отдельных бюджетов для отпуска, праздников или крупных покупок. Траты в проектах не смешиваются с ежедневными расходами.
*   **Недельный лимит:** Контроль "операционных" расходов с индикацией перерасхода.

### 💰 Планирование
*   **Калькулятор распределения:** Инструмент для деления входящего дохода на "корзины" (накопления, инвестиции, развлечения) по процентам.
*   **Настройка доходов:** Визуальные индикаторы дней зарплаты и аванса в календаре.

## 🏗 Структура проекта

*   `App.tsx`: Центральный хаб приложения. Управляет состоянием (React State), синхронизацией с `localStorage` и основной версткой.
*   `types.ts`: Определение интерфейсов данных (Project, Expense, Category и др.).
*   `services/aiService.ts`: Логика взаимодействия с Google GenAI SDK.
*   `components/`: Библиотека переиспользуемых UI-компонентов.
    *   `Calendar.tsx`: Сложный компонент навигации по датам.
    *   `AnalysisDashboard.tsx`: Визуализация структуры трат.
    *   `DailySpendingChart.tsx`: График динамики (Recharts).
    *   `ProjectSection.tsx`: Управление целевыми бюджетами.
    *   `CategorySettings.tsx`: Кастомизация списка категорий.

## 💾 Сущности данных (Data Entities)

### `Expense` (Трата)
Основная единица учета.
```typescript
{
  id: string;
  description: string; // "Кофе и круассан"
  amount: number;
  date: string;       // ISO формат
  category: string;   // Определяется AI или вручную
  emoji: string;      // Генерируется AI
  projectId?: string; // Привязка к конкретному событию
  isCategorizing: boolean; // Статус обработки нейросетью
}
```

### `Project` (Проект/Событие)
Изолированный бюджет.
```typescript
{
  id: string;
  name: string;
  budget: number;
  color: string;
  createdAt: string;
}
```

### `Category` (Категория)
Пользовательские настройки классификации.
```typescript
{
  id: string;
  name: string;
  emoji: string;
}
```

## 🛠 Технологический стек

*   **React 19**: UI-библиотека.
*   **Tailwind CSS**: Стилизация и адаптивность.
*   **Lucide React**: Пакет иконок.
*   **Recharts**: Библиотека для отрисовки столбчатых диаграмм.
*   **Google GenAI SDK (@google/genai)**: Интеграция с Gemini.
*   **LocalStorage API**: Локальное хранение данных пользователя (без бэкенда).

## ⚙️ Настройка для разработчика

1.  **API Ключ**: Для работы AI-функций требуется переменная окружения `process.env.API_KEY` с валидным ключом Google AI Studio.
2.  **Модель**: По умолчанию используется `gemini-3-flash-preview` для обеспечения минимальной задержки (latency) при категоризации.
3.  **Локализация**: Интерфейс полностью переведен на русский язык, календарь настроен на начало недели с понедельника.

## 📝 Заметки по разработке
При добавлении новых графиков рекомендуется использовать библиотеку `recharts`, так как она уже интегрирована. Для хранения тяжелых данных в будущем может потребоваться переход с `localStorage` на `IndexedDB`.
